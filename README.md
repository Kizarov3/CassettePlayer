# 📼 CassettePlayer
A retro-themed terminal music player featuring an animated ASCII cassette tape. Built with Python and Textual.

## 🚀 Quick Start
1. **Install dependencies:** `pip install -r requirements.txt`
2. **Run the player:** `python cassetteplayer.py`

## ⌨️ Controls
| Key | Action |
|---|---|
| `Space` | Play/Pause |
| `N` | Next Track |
| `Left/Right` | Seek 5s |
| `Q` | Quit |
